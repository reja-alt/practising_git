Rolling back previous stage command-> git checkout <commit no>

To show specific commit diff command -> git show <commit no>

To show git log in one commit command -> git log --oneline

Must show git diff before staging command -> git diff

To show differences between two commits command -> git diff <commit first> <commit second>

Inorder to show git diff after staging command -> git diff --staged

Automatic switch to branch after create that branch command -> git checkout -b <branch-name>

To delete any branch command -> git branch -D <branch-name>

To clean untracted file command -> git clean -f -n  and permanent delete use git clean -f
